Join NCI Proteomic Data Portal

NCI Proteomic Data Commons (PDC) is a pilot project is aimed to democratize access to cancer-related proteomic datasets as well as to provide sustainable computational support to the cancer research community. While most of the data in PDC is freely available with no restrictions, PDC will eventually host some controlled access data that requires authentication through NIH eRA Commons and dbGaP authorization. PDC also has a workspace component that currently allows users to submit data to PDC, store their mass spectrometry data and eventually to analyze the uploaded data.

Proteomics Data Commons FAQs
Is PDC launched?

The alpha-phase of the PDC launched in September 2018, is currently in beta phase, and anticipates going to full production in March 2020.

Can I submit data to PDC?

Yes, PDC now accepts Mass Spectrometry data from proteomic experiments specifically for data dependent and data independent acquisitions. You may contact nci.pdc.help@esacinc.com to request an account.

How do I download data?

Once you identify the files of your interest, move to ‘FIles’ tab and select one or all files. Then click ‘Download File Manifest’. The manifest will contain a download URL each selected file in a separate row. The URLs will expire after 48 hours.

Note: PDC is hosted on AWS cloud and is under active development. To reduce egress costs, not all files are currently available for download. However we will simplify the data download process in the coming months. If you are specifically interested in downloading CPTAC data, you may do so from CPTAC Data Portal.
